# N3MemoryCore (N3MC) - Free Edition

## 🚀 The Core Function
**N3MemoryCore (N3MC)** is a specialized **Persistent Memory and RAG System** designed for **Claude Code**.

Unlike standard LLM interactions, N3MC provides:
- **Cross-Session Memory:** It bridges the gap between different Claude Code sessions, allowing the AI to remember your project's context, and history.
- **Local Context Injection:** A seamless integration layer for RAG (Retrieval-Augmented Generation) that stays entirely within your local environment.
